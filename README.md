# Test Time Backend (FastAPI)

Простое тестовое API на FastAPI, которое возвращает текущее время и дату сервера.

## Что реализовано

- `GET /time` — возвращает текущее время сервера в формате ISO 8601 (UTC).
- `GET /date` — возвращает текущую дату сервера в UTC.
- `GET /date/local` — возвращает текущую локальную дату сервера и имя локального часового пояса.

### Примеры ответов

`GET /time`

```json
{
  "server_time": "2026-02-16T12:34:56.789012+00:00",
  "timezone": "UTC"
}
```

`GET /date`

```json
{
  "server_date": "2026-02-16",
  "timezone": "UTC"
}
```

`GET /date/local`

```json
{
  "server_date": "2026-02-17",
  "timezone": "Pacific/Auckland"
}
```

## Структура проекта

- `main.py` — FastAPI-приложение и маршруты
- `requirements.txt` — зависимости проекта
- `Dockerfile` — контейнеризация приложения
- `.gitignore` — игнорируемые Git-файлы

## Быстрый старт

1. Создайте и активируйте виртуальное окружение (рекомендуется).
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Запустите сервер разработки:

```bash
uvicorn main:app --reload
```

После запуска API будет доступно по адресу:

- `http://127.0.0.1:8000/time`
- `http://127.0.0.1:8000/date`
- `http://127.0.0.1:8000/date/local`

Swagger UI:

- `http://127.0.0.1:8000/docs`

ReDoc:

- `http://127.0.0.1:8000/redoc`

## Примечание

- Корневой маршрут `/` не настроен, поэтому запрос на `http://127.0.0.1:8000/` возвращает `404 Not Found` — это ожидаемое поведение для текущей версии.
- Если порт `8000` уже занят, запустите приложение на другом порту, например:

```bash
uvicorn main:app --reload --port 8001
```
